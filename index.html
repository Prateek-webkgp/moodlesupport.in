Grade Lookup & Reporting Tool

Faculty at IIT Patna were spending hours per week manually pulling grades across multiple courses with no central view. I built a PHP-based grade lookup system with role-based access, enabling faculty and admins to view multi-course grades from a single interface.

Tech: PHP, MySQL, Moodle APIs, role-based access control

Eliminated manual reporting dependency

Transcript Viewer (Custom Moodle Block)

Academic teams needed fast, filtered access to student transcripts across cohorts and batches — a feature not available in standard Moodle. I designed and deployed a custom Moodle block with cohort and batch filters, accessible to authorised roles only.

Tech: Moodle block API, PHP, cohort management

Streamlined academic & admin workflows

Infrastructure Automation & Reliability

Inherited a manual backup process prone to failures and storage bloat. Re-architected the entire backup and restore pipeline using shell scripts and cron jobs on Ubuntu LAMP servers, with DB tuning and uptime monitoring.

Tech: Ubuntu, Apache, MariaDB, Shell scripting, Cron

30% reduction in backup & restore time

BigBlueButton Live Session Integration

Tech: BigBlueButton, Moodle BBB plugin, server config
